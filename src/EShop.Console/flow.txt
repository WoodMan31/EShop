Поток:

1. Регистрация или авторизация
    1.1 Регистрация
        1.1.1 Запросить имя, электронную почту, пароль, адрес, карты
        1.1.2 Проверить, не зарегистрирован ли уже адрес электронной почты.
              Если адрес электронной почты уже зарегистрирован, показать пользователю ошибку.
        1.1.3 Создать пользователя и добавить его в базу данных.
        1.1.4 Авторизовать пользователя и показать сообщение об успехе.

    1.2 Авторизация
        1.2.1 Запросить электронную почту, пароль
        1.2.2 Проверить, правильны ли электронная почта и пароль.
              Если пароль неверен, показать пользователю ошибку.
        1.2.3 Авторизовать пользователя и показать сообщение об успехе.
    
2. Показать основное меню.
   Основное меню должно содержать следующие элементы:
   - Список товаров
   - Корзина покупок
   - Профиль

   2.1 Список товаров
       Это меню должно отображать список товаров (список названий).
       При нажатии на товар приложение должно отображать детали товара.
       
       2.1.1 Детали товара
             На этой странице должны отображаться данные о товаре: название, цена, описание, доступное количество.
             Пользователь должен иметь возможность добавить товар в корзину.

   2.2 Корзина покупок
       Это меню должно содержать следующие элементы:
       - Список элементов
       - Сделать заказ

       2.2.1 Список элементов
             На этой странице должен отображаться список элементов.
             При нажатии на элемент приложение должно отображать детали товара и предоставить возможность:
             - изменить количество (должно учитывать доступное количество)
             - удалить элемент из корзины
       
       2.2.2 Сделать заказ
             На этой странице должна отображаться общая стоимость заказа.
             Пользователь должен иметь возможность выбрать способ оплаты и сделать заказ.
             После оформления заказа приложение должно удалить элементы из корзины и 
             уменьшить доступное количество товаров.

   2.3 Профиль
       На этой странице должны отображаться данные пользователя: имя, электронная почта, адрес.
